#Sobre o projeto

O site exibe uma lista pré-carregada de animes com nome, descrição, número de episódios e imagem.
É possível marcar animes como concluídos ou para assistir, alternando entre páginas com o estado salvo via LocalStorage.

#Funcionalidades

-Lista de animes já carregada
-Exibição de capa, episódios e descrição
-Marcar como concluído
-Página exclusiva para animes concluídos
-Busca dinâmica
-Layout responsivo

#Tecnologias

HTML • CSS • JavaScript • JSON • LocalStorage • Google AI (Gemini)
